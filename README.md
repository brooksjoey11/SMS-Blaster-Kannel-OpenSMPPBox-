# SMS-Blaster-Kannel-OpenSMPPBox-
A high-throughput SMS blasting infrastructure built on Ubuntu 22.04 using Kannel and OpenSMPPBox. Configured to send 500+ SMS per second via SMPP carrier connections, with HTTP-based number ingestion, systemd service management, and built-in queue handling. Ideal for bulk messaging, alerts, or carrier-grade SMS deployments.
